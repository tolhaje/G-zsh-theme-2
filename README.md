Tolhaje's ZSH theme v2
=====================

![zsh theme](http://s9.postimg.org/fv5qru7hr/d_J_g67_YQCL8w_XEMBs_Zw_Oikb1hoqb5_V298_Q47_Nx_Wo8u_M.png)

An improved version minimal (https://github.com/Granze/G-zsh-theme) ZSH theme with git integration
Best to use with Solarized Dark terminal theme

If you are using [Antigen](https://github.com/zsh-users/antigen), just add the following line to your .zshrc

`antigen theme tolhaje/G-zsh-theme-2 granze2`
